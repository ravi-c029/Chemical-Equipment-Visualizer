**Chemical Equipment Parameter Visualizer** is a **hybrid analytics application** that transforms raw chemical equipment CSV data into meaningful, interactive visualizations — available as both a **browser-based web dashboard** AND a **native desktop application**.

```python
# ⚗️ SYSTEM PARAMETERS
input_data  = "CSV file → Flowrate, Pressure, Temperature readings"
processing  = "Django REST API + Pandas analytics engine 🐼"
output = {
    "🌐 web"    : "Interactive Chart.js visualizations in browser",
    "🖥️ desktop": "Native Matplotlib plots in PyQt5 window",
    "📊 stats"  : ["Avg Flowrate", "Avg Pressure", "Avg Temp", "Total Count"]
}
tagline = "One backend. Two frontends. Built for engineers. 🚀"
```

### ✅ Prerequisites

```bash
python --version    # 3.8+ required
node --version      # 16+ recommended
npm --version       # Comes with Node.js
```

---

### 🟠 Step 1 — Backend (Django)

```bash
# 📁 Navigate to backend
cd backend

# 🐍 Create virtual environment
python -m venv venv

# ▶️ Activate — Windows
venv\Scripts\activate
# ▶️ Activate — Mac/Linux
source venv/bin/activate

# 📦 Install dependencies
pip install django djangorestframework pandas django-cors-headers

# 🗃️ Apply migrations
python manage.py makemigrations
python manage.py migrate

# 🚀 Start server
python manage.py runserver
```
```
🟢 Backend → http://127.0.0.1:8000/
```

---

### 🔵 Step 2A — Web App (React)

> Open a **new terminal**

```bash
cd frontend-web
npm install axios chart.js react-chartjs-2 bootstrap
npm start
```
```
🟢 Web App → http://localhost:3000/
```

---

### 🟣 Step 2B — Desktop App (PyQt5)

> Open another **new terminal**

```bash
cd frontend-desktop
pip install PyQt5 matplotlib requests
python main.py
```
```
🟢 Desktop GUI window appears ✓
```

---

### ⚡ Quick Start (All at once)

```bash
# 🔴 Terminal 1 — Backend (start this FIRST)
cd backend && python manage.py runserver

# 🔵 Terminal 2 — Web interface
cd frontend-web && npm start

# 🟣 Terminal 3 — Desktop (optional)
cd frontend-desktop && python main.py
```
